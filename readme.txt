---------------------项目环境配置------------------------------
在电脑上进行Java配置--JDK开发环境搭建及环境变量配置（jdk1.8）
请下载并使用JetBrains的IntelliJ IDEA，在IDEA的软件页面中打开该项目，即LwgOnline目录即可
IDEA需要配置maven，需要配置tomcat做为Web容器，请下载并配置
本人使用的是apache-maven-3.6.0和apache-tomcat-7.0.42


--------------------数据库配置--------------------------------
在运行项目之前，需要先配置数据库，否则会没有数据可以查询，数据库文件为onlinetest.sql，在项目目录下
该文件可以在数据库软件中直接运行，如navicat，但运行前需要先创建数据库，库名为onlinetest


------------------------项目运行--------------------------------
在IDEA中打开的项目里，对src/main/resources下的jdbc.properties配置文件进行修改，将数据库的名字，
账户密码设置为你的数据库名字、账户、密码

然后点击本地的tomcat运行即可
